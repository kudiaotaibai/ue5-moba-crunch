## 服务器启动流程

### 服务器是怎么启动的？

**回顾协调器的启动命令：**

```bash
UnrealEditor.exe Crunch.uproject \
  -server \
  -SESSION_NAME="赵云飞的房间" \
  -SESSION_SEARCH_ID="abc-123" \
  -PORT=7777
```

**参数解释：**
- **-server：** 以服务器模式启动（不显示画面）
- **-SESSION_NAME：** 会话名称
- **-SESSION_SEARCH_ID：** 会话搜索ID
- **-PORT：** 服务器端口

### 服务器初始化：Init()

**文件位置：** `Source/Crunch/Private/Framework/CGameInstance.cpp:22`

```cpp
void UCGameInstance::Init()
{
    Super::Init();
    
    // 1. 检查是否在编辑器中
    if (GetWorld()->IsEditorWorld())
    {
        return;  // 编辑器中不执行网络初始化
    }

    // 2. 检查是否是服务器
    if (UCNetStatics::IsSessionServer(this))
    {
        CreateSession();  // 创建会话
    }
}
```

**逐步解析：**

#### 第1步：检查编辑器环境

```cpp
if (GetWorld()->IsEditorWorld())
{
    return;
}
```

- **IsEditorWorld()：** 检查是否在编辑器中运行
- **作用：** 编辑器中不需要创建网络会话
- **通俗解释：** 就像在家里练习，不需要连接服务器

#### 第2步：检查是否是服务器

```cpp
if (UCNetStatics::IsSessionServer(this))
{
    CreateSession();
}
```

- **IsSessionServer()：** 检查是否是专用服务器
- **CreateSession()：** 创建 EOS 会话

**IsSessionServer() 实现：**

```cpp
bool UCNetStatics::IsSessionServer(const UObject* WorldContextObject)
{
    return WorldContextObject->GetWorld()->GetNetMode() == ENetMode::NM_DedicatedServer;
}
```

- **GetNetMode()：** 获取网络模式
- **ENetMode::NM_DedicatedServer：** 专用服务器模式

**网络模式类型：**
- **NM_Standalone：** 单机模式（不联网）
- **NM_DedicatedServer：** 专用服务器（只跑逻辑，不显示画面）
- **NM_ListenServer：** 监听服务器（既是服务器又是客户端）
- **NM_Client：** 客户端

## 创建 EOS 会话

### 代码详解：CreateSession()

**文件位置：** `Source/Crunch/Private/Framework/CGameInstance.cpp:459`

```cpp
void UCGameInstance::CreateSession()
{
    // 1. 获取会话接口
    IOnlineSessionPtr SessionPtr = UCNetStatics::GetSessionPtr();
    if (SessionPtr)
    {
        // 2. 从命令行参数获取会话信息
        ServerSessionName = UCNetStatics::GetSessionNameStr();
        FString SessionSearchId = UCNetStatics::GetSessionSearchIdStr();
        SessionServerPort = UCNetStatics::GetSessionPort();
        
        UE_LOG(LogTemp, Warning, TEXT("###Create Session Name:%s, ID:%s, Port:%d"), 
               *ServerSessionName, *SessionSearchId, SessionServerPort);

        // 3. 生成会话设置
        FOnlineSessionSettings OnlineSessionSettings = 
            UCNetStatics::GenerateOnlineSessionSettings(
                FName(ServerSessionName),
                SessionSearchId,
                SessionServerPort
            );
        
        // 4. 注册创建完成回调
        SessionPtr->OnCreateSessionCompleteDelegates.RemoveAll(this);
        SessionPtr->OnCreateSessionCompleteDelegates.AddUObject(
            this, &UCGameInstance::OnSessionCreated
        );
        
        // 5. 创建会话
        if (!SessionPtr->CreateSession(0, FName(ServerSessionName), OnlineSessionSettings))
        {
            UE_LOG(LogTemp, Warning, TEXT("Session Creation Failed!!!!!"));
            SessionPtr->OnCreateSessionCompleteDelegates.RemoveAll(this);
            TerminateSessionServer();  // 创建失败，关闭服务器
        }
    }
    else
    {
        UE_LOG(LogTemp, Warning, TEXT("Can't Find sessionptr"));
        TerminateSessionServer();
    }
}
```

**逐步解析：**

#### 第1步：获取会话接口

```cpp
IOnlineSessionPtr SessionPtr = UCNetStatics::GetSessionPtr();
```

**GetSessionPtr() 实现：**

```cpp
IOnlineSessionPtr UCNetStatics::GetSessionPtr()
{
    IOnlineSubsystem* OnlineSubSystem = IOnlineSubsystem::Get();
    if (OnlineSubSystem)
    {
        return OnlineSubSystem->GetSessionInterface();
    }
    return nullptr;
}
```

- **IOnlineSubsystem：** 在线子系统接口
- **Get()：** 获取当前的在线子系统（EOS）
- **GetSessionInterface()：** 获取会话管理接口

**通俗解释：**
```
IOnlineSubsystem = EOS 的总管
GetSessionInterface() = 找到负责会话管理的部门
```

#### 第2步：获取命令行参数

```cpp
ServerSessionName = UCNetStatics::GetSessionNameStr();
FString SessionSearchId = UCNetStatics::GetSessionSearchIdStr();
SessionServerPort = UCNetStatics::GetSessionPort();
```

**GetSessionNameStr() 实现：**

```cpp
FString UCNetStatics::GetSessionNameStr()
{
    return GetCommandlineArgAsString(GetSessionNameKey());
}

FString UCNetStatics::GetCommandlineArgAsString(const FName& ParamName)
{
    FString OutVal = "";
    FString CommandlineArg = FString::Printf(TEXT("%s="), *(ParamName.ToString()));
    FParse::Value(FCommandLine::Get(), *CommandlineArg, OutVal);
    return OutVal;
}
```

- **FCommandLine::Get()：** 获取启动时的命令行参数
- **FParse::Value()：** 解析参数值
- **举例：**
  ```
  命令行：-SESSION_NAME="赵云飞的房间"
  ParamName：SESSION_NAME
  返回值："赵云飞的房间"
  ```

**通俗解释：**
```
就像读取配置文件
从启动命令中提取参数
```

#### 第3步：生成会话设置

```cpp
FOnlineSessionSettings OnlineSessionSettings = 
    UCNetStatics::GenerateOnlineSessionSettings(
        FName(ServerSessionName),
        SessionSearchId,
        SessionServerPort
    );
```

**GenerateOnlineSessionSettings() 详解：**

**文件位置：** `Source/Crunch/Private/Network/CNetStatics.cpp:7`

```cpp
FOnlineSessionSettings UCNetStatics::GenerateOnlineSessionSettings(
    const FName& SessionName,
    const FString& SessionSearchId,
    int Port
)
{
    FOnlineSessionSettings OnlineSessionSettings{};
    
    // 基础设置
    OnlineSessionSettings.bIsLANMatch = false;  // 不是局域网
    OnlineSessionSettings.NumPublicConnections = GetPlayerCountPerTeam() * 2;  // 最大玩家数：5*2=10
    OnlineSessionSettings.bShouldAdvertise = true;  // 公开会话，可被搜索
    OnlineSessionSettings.bUsesPresence = false;  // 不使用在线状态
    OnlineSessionSettings.bAllowJoinViaPresence = false;  // 不允许通过在线状态加入
    OnlineSessionSettings.bAllowJoinViaPresenceFriendsOnly = false;  // 不限制好友
    OnlineSessionSettings.bAllowInvites = true;  // 允许邀请
    OnlineSessionSettings.bAllowJoinInProgress = false;  // 不允许游戏中途加入
    OnlineSessionSettings.bUseLobbiesIfAvailable = false;  // 不使用大厅
    OnlineSessionSettings.bUseLobbiesVoiceChatIfAvailable = false;  // 不使用大厅语音
    OnlineSessionSettings.bUsesStats = true;  // 使用统计数据

    // 自定义设置
    OnlineSessionSettings.Set(
        GetSessionNameKey(),  // "SESSION_NAME"
        SessionName.ToString(),  // "赵云飞的房间"
        EOnlineDataAdvertisementType::ViaOnlineServiceAndPing  // 通过在线服务和Ping广播
    );
    
    OnlineSessionSettings.Set(
        GetSessionSearchIdKey(),  // "SESSION_SEARCH_ID"
        SessionSearchId,  // "abc-123"
        EOnlineDataAdvertisementType::ViaOnlineServiceAndPing
    );
    
    OnlineSessionSettings.Set(
        GetPortKey(),  // "PORT"
        Port,  // 7777
        EOnlineDataAdvertisementType::ViaOnlineServiceAndPing
    );
    
    return OnlineSessionSettings;
}
```

**重要参数详解：**

```cpp
OnlineSessionSettings.bIsLANMatch = false;
```
- **作用：** 是否局域网匹配
- **false：** 通过 EOS 在线匹配
- **true：** 只在局域网内可见

```cpp
OnlineSessionSettings.NumPublicConnections = GetPlayerCountPerTeam() * 2;
```
- **作用：** 最大公开连接数
- **GetPlayerCountPerTeam()：** 每队玩家数（5人）
- **计算：** 5 * 2 = 10人（两队）

```cpp
OnlineSessionSettings.bShouldAdvertise = true;
```
- **作用：** 是否公开会话
- **true：** 其他玩家可以搜索到
- **false：** 只能通过邀请加入

```cpp
OnlineSessionSettings.bAllowJoinInProgress = false;
```
- **作用：** 是否允许游戏中途加入
- **false：** 游戏开始后不能加入
- **true：** 随时可以加入

**自定义设置：**

```cpp
OnlineSessionSettings.Set(Key, Value, AdvertisementType);
```

- **Key：** 设置的键名
- **Value：** 设置的值
- **AdvertisementType：** 广播类型
  - **ViaOnlineServiceAndPing：** 通过在线服务和Ping广播（最常用）
  - **ViaOnlineService：** 只通过在线服务
  - **DontAdvertise：** 不广播

**通俗解释：**
```
Set() 就像给会话贴标签
SESSION_NAME = "赵云飞的房间"  ← 房间名标签
SESSION_SEARCH_ID = "abc-123"  ← 房间ID标签
PORT = 7777                    ← 端口号标签

ViaOnlineServiceAndPing 表示：
"把这些标签告诉 EOS，让别人能搜索到"
```

if (!SessionPtr->CreateSession(0, FName(ServerSessionName), OnlineSessionSettings))

- **CreateSession()：** 创建 EOS 会话

### 代码详解：OnSessionCreated()

```cpp
void UCGameInstance::OnSessionCreated(FName SessionName, bool bWasSuccessful)
{
    if (bWasSuccessful)
    {
        UE_LOG(LogTemp, Warning, TEXT("--------------------Session Created"));
        
        // 1. 设置超时定时器
        GetWorld()->GetTimerManager().SetTimer(
            WaitPlayerJoinTimeoutHandle,
            this,
            &UCGameInstance::WaitPlayerJoinTimeoutReached,
            WaitPlayerJoinTimeOutDuration  // 60秒
        );
        
        // 2. 加载大厅地图
        LoadLevelAndListen(LobbyLevel);
    }
    else
    {
        UE_LOG(LogTemp, Warning, TEXT("Session Creation Failed!!!!!"));
        TerminateSessionServer();
    }

    // 3. 清理回调
    if (IOnlineSessionPtr SessionPtr = UCNetStatics::GetSessionPtr())
    {
        SessionPtr->OnCreateSessionCompleteDelegates.RemoveAll(this);
    }
}
```

**流程：**

```
会话创建成功
    ↓
设置60秒超时定时器
（如果60秒内没人加入，关闭服务器）
    ↓
加载大厅地图
（服务器开始监听，等待玩家连接）
```

### 代码详解：LoadLevelAndListen()

```cpp
void UCGameInstance::LoadLevelAndListen(TSoftObjectPtr<UWorld> Level)
{
    // 1. 获取地图路径
    const FName LevelURL = FName(*FPackageName::ObjectPathToPackageName(Level.ToString()));
    
    if (LevelURL != "")
    {
        // 2. 构建旅行字符串
        FString TravelStr = FString::Printf(
            TEXT("%s?listen?port=%d"),
            *LevelURL.ToString(),
            SessionServerPort
        );
        
        UE_LOG(LogTemp, Log, TEXT("Server Traveling to: %s"), *TravelStr);
        
        // 3. 服务器旅行
        GetWorld()->ServerTravel(TravelStr);
    }
}
```

**参数解释：**

```cpp
TSoftObjectPtr<UWorld> Level
```
- **类型：** 软引用对象指针
- **作用：** 指向一个地图资源
- **举例：** `LobbyLevel` = "/Game/Maps/Lobby/Lobby"

**旅行字符串：**

```cpp
FString TravelStr = FString::Printf(TEXT("%s?listen?port=%d"), ...);
```

- **格式：** `地图路径?listen?port=端口号`
- **举例：** `/Game/Maps/Lobby/Lobby?listen?port=7777`
- **?listen：** 表示以监听模式加载（服务器模式）
- **?port=7777：** 指定监听端口

**ServerTravel()：**

```cpp
GetWorld()->ServerTravel(TravelStr);
```

- **作用：** 服务器旅行到新地图
- **效果：** 
  1. 卸载当前地图
  2. 加载新地图
  3. 开始监听指定端口
  4. 等待客户端连接

**通俗解释：**
```
ServerTravel 就像搬家
服务器从一个地图搬到另一个地图
?listen 表示"开门营业"，等客人来
```

## 玩家加入处理

### 代码详解：PlayerJoined()

```cpp
void UCGameInstance::PlayerJoined(const FUniqueNetIdRepl& UniqueId)
{
    // 1. 清除超时定时器
    if (WaitPlayerJoinTimeoutHandle.IsValid())
    {
        GetWorld()->GetTimerManager().ClearTimer(WaitPlayerJoinTimeoutHandle);
    }
    
    // 2. 记录玩家
    PlayerRecord.Add(UniqueId);
}
```

**参数：**

```cpp
const FUniqueNetIdRepl& UniqueId
```
- **类型：** 可复制的唯一网络ID
- **作用：** 玩家的唯一标识
- **用途：** 用来区分不同的玩家

**流程：**

```
玩家连接到服务器
    ↓
CGameSession::RegisterPlayer() 被调用
    ↓
调用 PlayerJoined(UniqueId)
    ↓
清除超时定时器（有人来了，不用关服务器了）
    ↓
记录玩家ID到 PlayerRecord 集合
```

### 代码详解：PlayerLeft()

```cpp
void UCGameInstance::PlayerLeft(const FUniqueNetIdRepl& UniqueId)
{
    // 1. 从记录中移除玩家
    PlayerRecord.Remove(UniqueId);
    
    // 2. 检查是否所有玩家都离开了
    if (PlayerRecord.Num() == 0)
    {
        UE_LOG(LogTemp, Error, TEXT("Player ALL Left"));
        TerminateSessionServer();  // 关闭服务器
    }
}
```

**流程：**

```
玩家断开连接
    ↓
CGameSession::UnregisterPlayer() 被调用
    ↓
调用 PlayerLeft(UniqueId)
    ↓
从 PlayerRecord 中移除玩家
    ↓
检查还有玩家吗？
    ├─ 有 → 继续运行
    └─ 没有 → 关闭服务器
```

**为什么要关闭服务器？**
```
没人玩了，服务器还开着浪费资源
就像商店没客人了，关门休息
```

### 代码详解：TerminateSessionServer()

```cpp
void UCGameInstance::TerminateSessionServer()
{
    if (IOnlineSessionPtr SessionPtr = UCNetStatics::GetSessionPtr())
    {
        // 1. 注册结束会话回调
        SessionPtr->OnEndSessionCompleteDelegates.RemoveAll(this);
        SessionPtr->OnEndSessionCompleteDelegates.AddUObject(
            this, &UCGameInstance::EndSessionCompleted
        );
        
        // 2. 结束会话
        if (!SessionPtr->EndSession(FName{ServerSessionName}))
        {
            // 结束失败，直接退出
            FGenericPlatformMisc::RequestExit(false);
        }
        else
        {
            // 结束成功，也退出（在回调中处理）
            FGenericPlatformMisc::RequestExit(false);
        }
    }
}

void UCGameInstance::EndSessionCompleted(FName SessionName, bool bWasSuccessful)
{
    // 退出程序
    FGenericPlatformMisc::RequestExit(false);
}
```

**FGenericPlatformMisc::RequestExit()：**
- **作用：** 请求退出程序
- **参数：** `false` - 不强制退出（正常退出）

## 网络工具类详解

### CNetStatics 类的作用

**通俗解释：**
```
CNetStatics 就像一个"工具箱"
里面装了很多常用的网络工具函数
需要什么工具，就从里面拿
```

### 常用函数详解

#### 1. 获取协调器 URL

```cpp
FString UCNetStatics::GetCoordinatorURL()
{
    // 1. 先从命令行参数获取
    FString CoordinatorURL = GetCommandlineArgAsString(GetCoordinatorURLKey());
    if (CoordinatorURL != "")
    {
        return CoordinatorURL;
    }

    // 2. 如果没有，从配置文件获取
    return GetDefaultCoordinatorURL();
}

FString UCNetStatics::GetDefaultCoordinatorURL()
{
    FString CoordinatorURL = "";
    GConfig->GetString(
        TEXT("Crunch.Net"),  // 配置节
        TEXT("CoordinatorURL"),  // 配置键
        CoordinatorURL,  // 输出变量
        GGameIni  // 配置文件（DefaultGame.ini）
    );
    UE_LOG(LogTemp, Warning, TEXT("CoordinatorURL: %s"), *CoordinatorURL);
    return CoordinatorURL;
}
```

**配置文件示例：**

**文件位置：** `Config/DefaultGame.ini`

```ini
[Crunch.Net]
CoordinatorURL=http://localhost:7777
```

**优先级：**
```
命令行参数 > 配置文件
```

**举例：**
```bash
# 使用命令行参数
UnrealEditor.exe -COORDINATOR_URL="http://192.168.1.100:7777"

# 使用配置文件
# 在 DefaultGame.ini 中设置
```

#### 2. 替换 URL 端口

```cpp
void UCNetStatics::ReplacePort(FString& OutURLStr, int NewPort)
{
    // 1. 解析 URL
    FURL URL(nullptr, *OutURLStr, TRAVEL_Absolute);
    
    // 2. 替换端口
    URL.Port = NewPort;
    
    // 3. 转回字符串
    OutURLStr = URL.ToString();
}
```

**举例：**
```cpp
FString URL = "192.168.1.100:7777";
ReplacePort(URL, 7778);
// 结果：URL = "192.168.1.100:7778"
```

## 完整服务器流程

```
第1步：协调器启动服务器进程
    ↓
第2步：服务器初始化
UCGameInstance::Init()
    ↓
第3步：检查是否是服务器模式
IsSessionServer() → true
    ↓
第4步：创建 EOS 会话
CreateSession()
    ├─ 从命令行获取参数
    ├─ 生成会话设置
    └─ 调用 EOS API 创建会话
    ↓
第5步：会话创建成功
OnSessionCreated()
    ├─ 设置60秒超时定时器
    └─ 加载大厅地图
    ↓
第6步：服务器开始监听
LoadLevelAndListen()
    └─ ServerTravel("Lobby?listen?port=7777")
    ↓
第7步：等待玩家连接...
    ↓
第8步：玩家连接成功
PlayerJoined()
    ├─ 清除超时定时器
    └─ 记录玩家ID
    ↓
第9步：游戏进行中...
    ↓
第10步：玩家离开
PlayerLeft()
    ├─ 移除玩家记录
    └─ 检查是否所有玩家都离开
    ↓
第11步：所有玩家都离开
TerminateSessionServer()
    ├─ 结束 EOS 会话
    └─ 退出程序
```

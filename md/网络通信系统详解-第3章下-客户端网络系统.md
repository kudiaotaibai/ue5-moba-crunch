## HTTP 响应处理

### 回调函数：SessionCreationRequestCompleted()

**文件位置：** `Source/Crunch/Private/Framework/CGameInstance.cpp:113`

```cpp
void UCGameInstance::SessionCreationRequestCompleted(
    FHttpRequestPtr Request,
    FHttpResponsePtr Response,
    bool bConnectedSuccessfully,
    FGuid SessionSearchId
)
{
    // 1. 检查连接是否成功
    if (!bConnectedSuccessfully)
    {
        UE_LOG(LogTemp, Warning, TEXT("Connection responded with connection was not successful!"));
        return;
    }

    UE_LOG(LogTemp, Warning, TEXT("Connection responded with connection was successful!!!"));

    // 2. 检查 HTTP 状态码
    int32 ResponseCode = Response->GetResponseCode();
    if (ResponseCode != 200)
    {
        UE_LOG(LogTemp, Warning, TEXT("Response code is %i"), ResponseCode);
        return;
    }

    // 3. 解析响应内容
    FString ResponseStr = Response->GetContentAsString();
    TSharedPtr<FJsonObject> JsonObject;
    TSharedRef<TJsonReader<>> Reader = TJsonReaderFactory<>::Create(ResponseStr);
    int32 Port = 0;

    if (FJsonSerializer::Deserialize(Reader, JsonObject) && JsonObject.IsValid())
    {
        Port = JsonObject->GetIntegerField(*(UCNetStatics::GetPortKey().ToString()));
    }

    UE_LOG(LogTemp, Warning, TEXT("Connect to Coordinator Successfully and the session created is on port : %i"), Port);
    
    // 4. 开始搜索刚创建的会话
    StartFindingCreatedSession(SessionSearchId);
}
```

## 搜索会话详解

### 为什么需要搜索？

**问题场景：**
```
1. 客户端请求创建服务器
2. 协调器启动服务器进程
3. 服务器启动需要时间（10-30秒）
4. 服务器创建 EOS 会话
5. 客户端怎么知道会话创建好了？
```

**解决方案：**
```
客户端定时搜索会话
每隔1秒搜索一次
找到了就加入
超时60秒还没找到就放弃
```

### 代码详解：StartFindingCreatedSession()

```cpp
void UCGameInstance::StartFindingCreatedSession(const FGuid& SessionSearchId)
{
    // 1. 验证 SessionSearchId
    if (!SessionSearchId.IsValid())
    {
        UE_LOG(LogTemp, Warning, TEXT("Session Search Id is invalid, can't start finding"));
        return;
    }

    // 2. 停止所有其他搜索
    StopAllSessionFindings();
    
    UE_LOG(LogTemp, Warning, TEXT("Start Finding Created Session with Id:%s"), *(SessionSearchId.ToString()));
    
    // 3. 设置定时器，每隔1秒搜索一次
    GetWorld()->GetTimerManager().SetTimer(
        FindCreatedSessionTimerHandle,
        FTimerDelegate::CreateUObject(this, &UCGameInstance::FindCreatedSession, SessionSearchId),
        FindCreatedSessionSearchInterval,  // 1.0 秒
        true,  // 循环执行
        0.f    // 立即开始
    );

    // 4. 设置超时定时器，60秒后超时
    GetWorld()->GetTimerManager().SetTimer(
        FindCreatedSessionTimeoutTimerHandle,
        this,
        &UCGameInstance::FindCreatedSessionTimeout,
        FindCreatedSessionTimeoutDuration  // 60.0 秒
    );
}
```

**定时器详解：**

```cpp

```
- **GetWorld()：** 获取当前世界对象
- **GetTimerManager()：** 获取定时器管理器
- **SetTimer()：** 设置一个定时器

**参数解释：**

```cpp

```
- **类型：** FTimerHandle
- **作用：** 定时器句柄，用来控制定时器（停止、暂停等）

- **FTimerDelegate：** 定时器委托
- **CreateUObject()：** 创建 UObject 成员函数委托
- **参数：**
  - `this`：当前对象
  - `&UCGameInstance::FindCreatedSession`：要调用的函数
  - `SessionSearchId`：传递给函数的参数

```cpp

```
- **作用：** 定时器间隔
- **值：** 1.0 秒
- **意思：** 每隔1秒执行一次

```cpp

```
- **作用：** 是否循环
- **true：** 每隔1秒执行一次，直到手动停止

### 代码详解：FindCreatedSession()

```cpp
void UCGameInstance::FindCreatedSession(FGuid SessionSearchId)
{
    UE_LOG(LogTemp, Warning, TEXT("Trying to find created session"));
    
    // 1. 获取会话接口
    IOnlineSessionPtr SessionPtr = UCNetStatics::GetSessionPtr();
    if (!SessionPtr)
    {
        UE_LOG(LogTemp, Warning, TEXT("Session Pointer is invalid"));
        return;
    }

    // 2. 创建搜索对象
    SessionSearch = MakeShareable(new FOnlineSessionSearch);
    if (!SessionSearch)
    {
        UE_LOG(LogTemp, Warning, TEXT("Session Search Pointer is invalid"));
        return;
    }

    // 3. 配置搜索参数
    SessionSearch->bIsLanQuery = false;  // 不是局域网搜索
    SessionSearch->MaxSearchResults = 1;  // 最多返回1个结果
    SessionSearch->QuerySettings.Set(
        UCNetStatics::GetSessionSearchIdKey(),
        SessionSearchId.ToString(),
        EOnlineComparisonOp::Equals  // 精确匹配
    );

    // 4. 注册搜索完成回调
    SessionPtr->OnFindSessionsCompleteDelegates.RemoveAll(this);
    SessionPtr->OnFindSessionsCompleteDelegates.AddUObject(
        this, &UCGameInstance::FindCreateSessionCompleted
    );
    
    // 5. 执行搜索
    if (!SessionPtr->FindSessions(0, SessionSearch.ToSharedRef()))
    {
        UE_LOG(LogTemp, Warning, TEXT("Session Search failed"));
        SessionPtr->OnFindSessionsCompleteDelegates.RemoveAll(this);
    }
}
```

**搜索参数详解：**

```cpp
SessionSearch->bIsLanQuery = false;
```
- **作用：** 是否局域网搜索
- **false：** 通过 EOS 在线搜索
- **true：** 只搜索局域网内的会话

```cpp
SessionSearch->MaxSearchResults = 1;
```
- **作用：** 最多返回多少个结果
- **1：** 只返回1个（因为我们知道 SessionSearchId 是唯一的）

```cpp
SessionSearch->QuerySettings.Set(
    UCNetStatics::GetSessionSearchIdKey(),  // "SESSION_SEARCH_ID"
    SessionSearchId.ToString(),              // "123-456-789"
    EOnlineComparisonOp::Equals              // 精确匹配
);
```
- **作用：** 设置搜索条件
- **QuerySettings：** 查询设置
- **Set()：** 添加一个搜索条件
  - **Key：** SESSION_SEARCH_ID
  - **Value：** 我们的会话ID
  - **Op：** 比较操作符（等于）

**通俗解释：**
```
告诉 EOS："帮我找 SESSION_SEARCH_ID = '123-456-789' 的会话"
就像在数据库里查询：
SELECT * FROM Sessions WHERE SESSION_SEARCH_ID = '123-456-789'
```

### 代码详解：FindCreateSessionCompleted()

```cpp
void UCGameInstance::FindCreateSessionCompleted(bool bWasSuccessful)
{
    // 1. 检查搜索是否成功，且有结果
    if (!bWasSuccessful || SessionSearch->SearchResults.Num() == 0)
    {
        return;  // 没找到，继续等待下次搜索
    }

    // 2. 找到了！停止搜索
    StopFindingCreatedSession();
    
    // 3. 加入找到的会话
    JoinSessionWithSearchResult(SessionSearch->SearchResults[0]);
}
```

**流程：**
```
搜索完成回调
    ↓
检查是否找到会话？
    ├─ 没找到 → 返回，等待下次搜索
    └─ 找到了 → 停止搜索
                ↓
           加入会话
```

## 加入会话详解

### 代码详解：JoinSessionWithSearchResult()

```cpp
void UCGameInstance::JoinSessionWithSearchResult(const FOnlineSessionSearchResult& SearchResult)
{
    UE_LOG(LogTemp, Warning, TEXT("Join Session With Search Result"));
    
    // 1. 获取会话接口
    IOnlineSessionPtr SessionPtr = UCNetStatics::GetSessionPtr();
    if (!SessionPtr)
    {
        UE_LOG(LogTemp, Warning, TEXT("Session Pointer is invalid"));
        return;
    }

    // 2. 从搜索结果中提取信息
    FString SessionName = "";
    SearchResult.Session.SessionSettings.Get<FString>(
        UCNetStatics::GetSessionNameKey(),
        SessionName
    );
    
    const FOnlineSessionSetting* PortSetting = 
        SearchResult.Session.SessionSettings.Settings.Find(UCNetStatics::GetPortKey());
    int64 Port = 7777;
    PortSetting->Data.GetValue(Port);

    UE_LOG(LogTemp, Warning, TEXT("Trying to join session:%s, at port:%d"), *SessionName, Port);
    
    // 3. 注册加入完成回调
    SessionPtr->OnJoinSessionCompleteDelegates.RemoveAll(this);
    SessionPtr->OnJoinSessionCompleteDelegates.AddUObject(
        this, &UCGameInstance::JoinSessionCompleted, (int)Port
    );
    
    // 4. 执行加入操作
    if (!SessionPtr->JoinSession(0, FName(SessionName), SearchResult))
    {
        UE_LOG(LogTemp, Warning, TEXT("Session Join failed........"));
        SessionPtr->OnJoinSessionCompleteDelegates.RemoveAll(this);
        OnJoinSessionFailed.Broadcast();
    }
}
```

**参数详解：**

```cpp
const FOnlineSessionSearchResult& SearchResult
```
- **类型：** 会话搜索结果
- **包含：**
  - 会话ID

  - 会话设置（端口、玩家数等）
  - 服务器地址

**提取会话信息：**

- **作用：** 从会话设置中获取会话名称
- **Get<FString>()：** 获取字符串类型的值
- **结果：** SessionName = "赵云飞的房间"

```cpp

PortSetting->Data.GetValue(Port);
```

- **作用：** 获取端口号
- **Find()：** 查找设置项
- **GetValue()：** 获取值

### 代码详解：JoinSessionCompleted()

```cpp
void UCGameInstance::JoinSessionCompleted(
    FName SessionName,
    EOnJoinSessionCompleteResult::Type JoinResult,
    int Port
)
{
    // 1. 获取会话接口
    IOnlineSessionPtr SessionPtr = UCNetStatics::GetSessionPtr();
    if (!SessionPtr)
    {
        UE_LOG(LogTemp, Warning, TEXT("Join Session Completed, but can't find session pointer"));
        OnJoinSessionFailed.Broadcast();
        return;
    }

    // 2. 检查加入结果
    if (JoinResult == EOnJoinSessionCompleteResult::Success)
    {
        // 加入成功
        StopAllSessionFindings();
        UE_LOG(LogTemp, Warning, TEXT("Join session %s completed successfully, port:%d"), 
               *(SessionName.ToString()), Port);

        // 3. 获取服务器地址
        FString TravelURL = "";
        SessionPtr->GetResolvedConnectString(SessionName, TravelURL);

        // 4. 测试URL（可选）
        FString TestingURL = UCNetStatics::GetTestingURL();
        if (!TestingURL.IsEmpty())
        {
            TravelURL = TestingURL;
        }

        // 5. 替换端口号
        UCNetStatics::ReplacePort(TravelURL, Port);

        UE_LOG(LogTemp, Warning, TEXT("Traveling to Session at :%s"), *TravelURL);

        // 6. 客户端旅行到服务器
        GetFirstLocalPlayerController(GetWorld())->ClientTravel(TravelURL, TRAVEL_Absolute);
    }
    else
    {
        // 加入失败
        OnJoinSessionFailed.Broadcast();
    }

    // 7. 清理回调
    SessionPtr->OnJoinSessionCompleteDelegates.RemoveAll(this);
}
```

**加入结果类型：**

```cpp
EOnJoinSessionCompleteResult::Type
```
- **Success：** 加入成功
- **SessionIsFull：** 会话已满
- **SessionDoesNotExist：** 会话不存在
- **CouldNotRetrieveAddress：** 无法获取服务器地址
- **AlreadyInSession：** 已经在会话中
- **UnknownError：** 未知错误

**获取服务器地址：**

```cpp
FString TravelURL = "";
SessionPtr->GetResolvedConnectString(SessionName, TravelURL);
```
- **作用：** 获取服务器的连接地址
- **结果示例：** `192.168.1.100:7777`

**客户端旅行：**

```cpp
GetFirstLocalPlayerController(GetWorld())->ClientTravel(TravelURL, TRAVEL_Absolute);
```
- **ClientTravel()：** 客户端旅行到服务器
- **TravelURL：** 服务器地址
- **TRAVEL_Absolute：** 绝对旅行（断开当前连接，连接到新服务器）

**通俗解释：**
```
ClientTravel 就像"传送"
把你从当前地图传送到服务器的地图
TRAVEL_Absolute 表示完全断开重连
```
